This is a sample post included with the starter.

ConsolePress is intentionally small: Markdown files in, static HTML out. If you
want to make the site your own, start by editing `site.config.js`, then replace
the sample posts and pages with your real content.

The default theme leans into a terminal metaphor, but the templates are plain
HTML and the styling lives in a single CSS file, so it is easy to reshape.
